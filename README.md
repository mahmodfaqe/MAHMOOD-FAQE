
# BoKwe Multilingual Website

## Setup

1. Install dependencies:
    pip install -r requirements.txt

2. Run the app:
    python app.py

3. Visit:
    http://127.0.0.1:5000

## Deployment

- Ready for Railway, Heroku, or VPS deployment.
- Add 'web: gunicorn app:app' in Procfile for production.
